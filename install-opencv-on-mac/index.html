在 Mac 上安装 Opencv

在 Mac 安装 opencv，使用的方案和 linux 是一样的。

先在官网页面右侧下载OpenCV for Linux/Mac。

打开终端，用tar -xvf opencv压缩文件解压文件。

cd 进入解压后的文件夹。

mkdir release

cd release

cmake -D CMAKE_BUILD_TYPE=RELEASE -D CMAKE_INSTALL_PREFIX=/usr/local ..

make

sudo make install

依次执行，最后没报错就是成功了。

各种方法都试了，in vain.最后哥们儿帮我找到了答案：
